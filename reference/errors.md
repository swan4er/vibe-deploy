# Обработка ошибок

## Ошибки API Timeweb Cloud

| Код | Причина | Действие |
|-----|---------|----------|
| 401 | Неверный или истёкший токен | Попросить новый токен |
| 403 | Нет прав | Проверить разрешения токена |
| 404 | Ресурс не найден | Проверить ID ресурса |
| 409 | Конфликт (имя занято) | Изменить имя ресурса |
| 423 | Ресурс заблокирован | Подождать и повторить |
| 429 | Превышен лимит запросов | Подождать 5 секунд и повторить |
| 500 | Внутренняя ошибка | Повторить через 30 секунд |

## Ошибки деплоя

| Проблема | Диагностика | Решение |
|----------|-------------|---------|
| Сервер не создаётся | Проверить баланс, preset_id | Пополнить / другой пресет |
| SSH недоступен | Сервер ещё запускается | Подождать 1-3 минуты |
| Docker build fails | Ошибки в Dockerfile | Исправить, показать лог |
| Порт занят | Другой сервис | Изменить порт |
| БД не подключается | Неверный connection string | Проверить DATABASE_URL |
| SSL не выпускается | DNS не обновился | Подождать, проверить записи |
| App Platform падает | Ошибка сборки/запуска | Проверить логи, команды |
| Нехватка памяти | Мало RAM | Обновить пресет |
| Permission denied | Неверный SSH-ключ | Пересоздать ключ |

## Откат при критической ошибке

Предложи пользователю:
1. Попробовать исправить и продолжить
2. Удалить созданные ресурсы и начать заново
3. Оставить как есть — продолжить вручную через панель

Удаление ресурсов:
```bash
python3 scripts/timeweb_api.py delete-server --id <ID>
python3 scripts/timeweb_api.py delete-db --id <ID>
python3 scripts/timeweb_api.py delete-app --id <ID>
```
